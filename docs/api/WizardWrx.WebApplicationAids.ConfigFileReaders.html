GetConnectionStringFromWebConfig(String, String)

Get the connection string identified by name by string pstrConnectionStringName from the official connectionStrings section of the web.config file specified by the relative or absolute file named in string pstrAbsoluteConfigFileName.

Declaration
public static ConnectionStringSettings GetConnectionStringFromWebConfig(string pstrAbsoluteConfigFileName, string pstrConnectionStringName)
Parameters
Type Name Description
System.String pstrAbsoluteConfigFileName

This string must be either an absolute (fully qualified) file name or a name that is valid relative to the current working directory.
System.String pstrConnectionStringName

This string must contain the "name" property of a node in the connectionStrings section of a well-formed web.config file.
Returns
Type Description
System.Configuration.ConnectionStringSettings

If it succeeds, the return value is a ConnectionStringSettings object that has valid name and connectionString properties. Otherwise, the return value is NULL.

Its providerName property may have a value, or it may be null.

Regardless, the returned ConnectionStringSettings object can be treated as if it had been read from a web.config file in the usual way.

Remarks

This method works by treating the web.config file as a standard XML document.
